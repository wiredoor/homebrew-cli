# Wiredoor Cli

## How do I install these formulae?

`brew install wiredoor/cli/<formula>`

Or `brew tap wiredoor/cli` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "wiredoor/cli"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
